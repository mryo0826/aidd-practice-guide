# ユーザーメモリ テンプレート（`~/.claude/CLAUDE.md`）

> **このファイルの役割:** Claude Code のユーザーメモリ（`~/.claude/CLAUDE.md`）に登録する個人作業原則の正本。
> 全プロジェクトで共通適用される個人作業原則をここで管理し、プロジェクト固有の制約は各リポジトリの `CLAUDE.md` に置く。

---

## 適用方法

1. `~/.claude/CLAUDE.md` を開く（無ければ新規作成する）
2. 下記の「ユーザーメモリ本文」を追記する
3. Claude Code はこのファイルを全プロジェクトで常時ロードする

---

## ユーザーメモリ本文

以下を `~/.claude/CLAUDE.md` に追記する。「日本語で応答する」原則が未設定なら、それも含める:

```markdown
## 作業原則
- 日本語で応答する
- 調査 → 方針決定 → 実装の順序を守る。いきなりコードを書かない
- 既存コードのパターン・規約を最優先で踏襲する
- 変更前に影響範囲を確認し、必要最小限の修正に留める

## AI協調の原則
- 1つの長いセッションを続けず、トピック別にセッションを切る。確定内容はMarkdownに書き戻す
- AIの提案を鵜呑みにしない。人間が裁定者として機能する
- 異なるモデルでクロスレビューし、単一モデルのバイアスを相殺する

## Token効率
- Claude Code 内蔵ツール → CLI → MCP の優先順でツールを選ぶ
- MCPはCLI代替が存在しない場合のみ導入する
- コンテキストには必要最小限のファイルだけを渡す（@参照で絞る）
```
